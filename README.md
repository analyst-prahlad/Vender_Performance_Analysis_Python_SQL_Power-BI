# Vendor Performance Analysis

## Overview
This project analyzes vendor performance, sales, inventory turnover, and profitability using SQL, Python, and Power BI. The goal is to identify business opportunities, optimize vendor management, and support data-driven decision-making.

## Tools Used
- SQL
- Python
- Pandas
- Matplotlib
- Power BI
- Excel

## Key Insights
- Identified top-performing and low-performing vendors.
- Analyzed the impact of bulk purchasing on cost savings.
- Evaluated inventory turnover and unsold inventory.
- Compared profit margins across vendors.
- Generated actionable business recommendations.

## Dashboard
The Power BI dashboard provides insights into:
- Vendor Performance
- Sales & Profit Analysis
- Inventory Turnover
- Purchase Trends

## Business Recommendations
- Diversify vendor partnerships to reduce dependency risks.
- Optimize pricing for low-sales, high-margin products.
- Improve inventory management to reduce holding costs.
- Leverage bulk purchasing for cost savings.
